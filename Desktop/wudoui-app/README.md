WudoUI:

WudoUI is a responsive and visually appealing web application showcasing premium furniture, designed with React and styled with CSS. It features dynamic content, intuitive navigation, and a user-friendly interface.

Features
Responsive Design: Optimized for various screen sizes with a modern, clean aesthetic.
Navigation Bar: Easy navigation with links to Home, Furniture, Product, and Store.
Product Display: Highlights a premium chair with images, reviews, and customization options.

Dynamic Content:
Color Options: Interactively choose colors for the product.
Quantity Selector: Increase or decrease the number of items to purchase.
Interactive Components:
Action buttons like "Add to Cart" and "Buy Now."
Price and quantity adjustment functionality.
Custom Footer: Minimalistic footer for branding and additional links.

Technologies Used
React.js: Component-based architecture for building the user interface.
CSS: Styling with flexbox, grid, and custom designs, including gradients and animations.
FontAwesome: For icons used in navigation and product actions.
Project Structure

src/
  ├── WudoUI.js  # Main React component
  ├── WudoUI.css # CSS file for styling
  ├── index.js   # Entry point for React app
  └── assets/    # Assets folder for images and icons
  
Setup and Installation
Clone the Repository:

git clone https://github.com/eungobs/payment-button.git
cd wudoui-app
Install Dependencies: Ensure you have Node.js and npm installed, then run:

npm install
Run the Application: Start the development server with:

npm start
Open in Browser: Visit http://localhost:3000 to view the application.

Usage
Navigate through the website using the top navbar.
Explore the premium chair product, view its details, and customize your selection.
Use the action buttons to add the item to the cart or purchase directly.
Enjoy a seamless browsing experience with responsive layouts.
